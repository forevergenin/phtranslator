PhWordpad

PhWordpad allows you to phonetaically transliterate the words to the selected Indic Language. Go ahead and type something - once the word is complete, it will be transliterated automatically. Or you can select any text and press Ctrl+T to transcribe it.

Once you are done with editing, you can save the document in multiple platform compatible formats such as OpenOffice document, Html Document and PDF document. Just use the Save or Export options from the File Menu.
